# OPENCLAW — Document Governance Protocol
## Advisory Notes and System Documents

---

## SECTION 1 — PURPOSE

This document defines the role of advisory notes in the OpenClaw project
and establishes a clear boundary between advisory notes and system documents.

---

## SECTION 2 — ADVISORY NOTES

Advisory notes are reference documents. They capture thinking, analysis,
options, and recommendations for the operator's consideration.

Advisory notes do not trigger system changes. Approval of an advisory note
means the operator has read and accepted the content as useful reference.
It does not constitute an instruction to act.

Advisory notes never govern work. They are never cited as the authority for
a system change, a phase decision, or an implementation action.

---

## SECTION 3 — SYSTEM DOCUMENTS

System documents govern work. They define how the OpenClaw system operates,
what phase is active, what is in and out of scope, and what the standards are.

System documents are created, updated, or superseded only when the operator
explicitly instructs it in the session. The instruction must be direct — not
implied by, or derived from, an advisory note.

---

## SECTION 4 — THE BOUNDARY

| | Advisory Note | System Document |
|-|--------------|-----------------|
| Purpose | Reference — captures thinking and recommendations | Governs work |
| Triggers system changes? | No | Yes |
| Approval means | Operator has read and accepted as useful reference | Operator has authorised the content to govern work |
| Can be cited as authority for action? | No | Yes |

---

## SECTION 5 — HARD RULES

| Rule | Statement |
|------|-----------|
| DG-01 | Advisory notes are reference only. Approval of an advisory note does not authorise any system change. |
| DG-02 | System documents are updated only on explicit operator instruction in the session. |
| DG-03 | Claude CoWork does not incorporate advisory note content into system documents unless the operator explicitly instructs it. |
| DG-04 | No system document may cite an advisory note as its governing authority. |
